## ⚙️ How it works

The plugin sits between OpenCode and Claude auth. It uses your existing Claude Code sign-in flow instead of asking you to create a new one.

That means:

- OpenCode asks for auth
- The plugin passes that request to Claude Code credentials
- Claude Code handles the sign-in
- OpenCode receives access through the same account
